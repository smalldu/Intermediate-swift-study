

### 重命名本地git 

```
git branch -m <oldname> <newname>
```

如果是当前branch

```
git branch -m <newname>
```

获取远程某个分支
```
git fetch origin develop
```

```
git clone -b <branch> <remote_repo>
```

删除远程分支
```
git push origin --delete <branchName>
```
删除远程tag            

```
git push origin --delete tag <tagname>
```
